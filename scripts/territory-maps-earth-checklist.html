Como desenhar e exportar

  1. Abra o Google Earth Pro. Camada Estradas ligada.
  2. Crie a pasta Jardim Elizabeth em Lugares e mantenha todos os polígonos nela (para ver buracos/sobreposição).
  3. Para cada Txx: use o link Maps → no Earth, Adicionar polígono, nome Txx.
  4. Estilo: linha amarela largura 3–4; área amarela opacidade ~25–30%.
  5. Trace só o perímetro do cartão pelas ruas listadas — sem cortar quadra/casa.
  6. Vista de cima, norte para cima; enquadre o polígono; capture/exporte JPEG.
  7. Salve como img/territorios/tXX.jpg (800–1400px de largura, de preferência < 1,5 MB).
  8. Valide: node scripts/validate-territory-maps.mjs → depois peça commit/deploy.

Isto não substitui o mapa S-6 único da congregação para Betel — só os cartões do site.
